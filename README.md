# Overview

This project was to learn more about Javascript DOM manipulation and custom user-interface animations.
It has helped me learn about what Javascript I can use to adapt my website to my needs.

I built a custom multi-page web portfolio with a button to translate English text to a stylish Elvish
Tengwar script that works on any page.  I also added a hover box with the English description to allow
easier navigation while viewing in the Elvish script.  The last thing I added was an easter egg that 
shows a Matrix-styled background when you type cyber.

This project allowed me to explore Javascript implementation with existing HTML and CSS.
